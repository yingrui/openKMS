# openKMS

Open Knowledge Management System — documents, articles, and knowledge bases with RAG Q&A.

## Features

- **Documents** – Channel tree, upload PDF/HTML/ZIP/images → Markdown
- **Articles** – Channel tree, CMS-style content with fields
- **Knowledge Bases** – Add docs/articles, generate FAQs, semantic search & Q&A

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5174](http://localhost:5174)

## Tech

React 19, TypeScript, Vite, React Router, Lucide Icons
